1. GitHubリポジトリの作成

GitHubでウェブサイトを公開するには、まずリポジトリを作成する必要があります。以下の手順でリポジトリを作成しましょう。

ステップ1: GitHubにログイン

まず、GitHubのウェブサイト にログインします。

ステップ2: 新しいリポジトリを作成

ログイン後、画面の右上にある「+」アイコンをクリックし、「New repository」を選択します。

ステップ3: リポジトリの情報を入力

リポジトリの名前、説明（オプション）、公開設定を選択します。ウェブサイト用のリポジトリなので、「Repository name」には「username.github.io」と入力します（`username` はあなたのGitHubユーザー名に置き換えてください）。

ステップ4: リポジトリの初期設定

リポジトリの作成が完了したら、READMEファイルを作成するオプションが表示されますが、今回は「Add a README file」を選択せずに「Create repository」をクリックします。

2. ウェブサイトファイルのアップロード

リポジトリを作成したら、ウェブサイトのファイルをアップロードしていきます。

ステップ1: リポジトリにアクセス

作成したリポジトリにアクセスし、「Add file」ボタンをクリックして「Upload files」を選択します。

ステップ2: ファイルのアップロード

ローカルのウェブサイトファイル（`index.html`, `styles.css`, など）をドラッグ＆ドロップでアップロードします。すべてのファイルがアップロードできたら、「Commit changes」ボタンをクリックします。

ステップ3: GitHub Pagesの設定

リポジトリの「Settings」タブを開き、「Pages」セクションに移動します。「Source」ドロップダウンメニューから「main」ブランチを選択し、「Save」をクリックします。

ステップ4: ウェブサイトの公開

設定が完了すると、数分後にGitHub Pagesで公開されます。ウェブサイトのURLは「https://username.github.io」になります。ブラウザでアクセスして、ウェブサイトが正しく表示されることを確認しましょう。
